### Framework utilizado:
**Bootstrap 5.3 (CDN)**

### Razones de la elección:
- Permite construir un sitio web profesional y responsive de forma rápida.
- Incluye componentes dinámicos sin código JavaScript adicional.
- Favorece la accesibilidad y la coherencia visual.

### Mejoras implementadas:
- Navbar responsive con menú colapsable.
- Botón "Mostrar más información" con efecto *collapse*.
- Tarjeta informativa con efecto hover.
- Efecto hover también aplicado a la lista de los 3 aportes.
- Formulario validado visualmente (nombre, email y mensaje).
- Envío tipo GET para mostrar datos en la barra de direcciones.
- Colores y fuentes institucionales del INET aplicados con Bootstrap + CSS personalizado.
